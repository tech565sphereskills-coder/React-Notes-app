# 📝 React Notes App - Premium Edition

## 🌟 About The Project
The React Notes App is a modern, feature-rich web application that allows users to seamlessly create, edit, pin, search, and manage their personal notes. Originally designed with core functionalities in mind, this project has been completely overhauled to feature a premium, responsive, glassmorphic UI paired with smooth micro-animations.

With both Light and Dark mode options dynamically integrated, users can customize their note-taking experience to fit their aesthetic preferences while enjoying a highly functional and polished design.

## ✨ Key Features
- **Modern UI/UX:** Premium glassmorphism aesthetic featuring soft shadow lifts and interactive hover effects.
- **Theme Support:** Fully functional dynamic Light and Dark mode toggle with seamless CSS transitions.
- **In-Place Editing:** Create new notes (with a live character limit counter) and selectively edit existing notes directly on the dashboard.
- **Pin Important Notes:** Tap the pin icon to prioritize specific notes, guaranteeing they always remain at the top of your layout.
- **Color Customization:** Customize individual notes via an elegant built-in color-picker.
- **Smart Search/Filter:** Instantly filter your active notes by input characters in real-time.
- **Copy to Clipboard:** One-click utility copy functionality to instantly grab note text.
- **Persistent Storage:** All notes, custom colors, theme states, and pinned states are natively saved to your device's `localStorage`.

## 🛠 Technologies Used
- **[React.js](https://reactjs.org/) (v17)** - Core Frontend JavaScript Architecture
- **Vanilla CSS3** - Custom styling utilizing complex CSS Variables (for native theming) and modern Grid layouts
- **[React Icons](https://react-icons.github.io/react-icons/)** - Fast, lightweight, and scalable vector icons
- **[Nanoid](https://github.com/ai/nanoid)** - Secure and predictable unique ID generation 
- **Google Fonts (Inter)** - Clean and highly legible modern typography

## 🚀 Getting Started

To get a local setup up and running, follow these easy steps:

### Prerequisites
Ensure you have Node.js and npm (Node Package Manager) installed in your local environment.

### Installation
1. Clone your repository:
   ```bash
   git clone https://github.com/your-username/react-notes-app.git
   ```
2. Navigate into the project directory:
   ```bash
   cd react-notes-app
   ```
3. Install all dependencies:
   ```bash
   npm install
   ```
4. Start the Application Development Server:
   ```bash
   npm start
   ```
5. Open up your browser and navigate to `http://localhost:3000` to view the app!

*Note: For Node v24+ environments, `package.json` scripts have already been preemptively configured with `NODE_OPTIONS=--openssl-legacy-provider` to prevent any OpenSSL hashing build errors.*
